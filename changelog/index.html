Release version 1.8.1.0 (Github)

  • Added blacklist and whitelist filters to Property Transformer.
  • Added option to assign additions sampleset and custom index with Hitsound Preview Helper.
  • Added option to exit without saving the settings. Right-click the exit button.

  • Added Auto sampleset option to default sample in Hitsound Studio.

  • Fixed sliderend muting incorrectly detecting snap divisors.

Release version 1.8.0.0 (Github)

Release version 1.7.1.0 (Github)

  • All tools now support ALL beat snap divisors.
  • Improved the timestamp input converter, so it accepts more types of inputs.
  • Added midi sample export option in Hitsound Studio.
  • Hitsound Copier now allows an empty 'Copy from' field to use as an empty beatmap.
  • Sliderend muting now no longer mutes objects with hitsounds on them.
  • Added a cancel button to skip showing inner exceptions.
  • Optimized timing points querying operations for faster running times across all tools.
  • Fixed a minor inconsistency between osu! beatmap saving and Mapping Tools beatmap saving.

Release version 1.7.0.1 (Github)

  • Fixed Auto-fail Detector generating solutions which are impossible to place.
  • Fixed timeline in Auto-fail Detector not removing lines after running with less showing options checked.

Release version 1.7.0.0 (Github)

  • Added Auto-fail Detector.
  • Updated About page for all contributors and supporters.

Release version 1.6.8.1 (Github)

  • Red lines in Sliderator now have snapping.
  • Mapping Tools window position now gets retained on restart.
  • Geometry Dashboard now no longer responds to hotkeys while osu! is deactivated.
  • Fixed snapping in Geometry Dashboard sometimes getting confused.
  • Fixed the 'Remove unclickable hitsounds' function from Map Cleaner.
  • Improved success rate of 'Get current beatmap'.
  • Fixed crash when trying to run a tool with an invalid path.
  • Made it less likely to crash on startup without any error showing.
  • Fixed the filenames of automatic backup files.
  • Fixed slidertick/sliderslide copy when copying to multiple maps at the same time in Hitsound Copier.
  • Fixed slidertick/sliderslide copy trying to convert for invalid samples.
  • Fixed tags field in Metadata Manager being able to remove duplicate tags while you're still typing.
  • Fixed error when trying to run Timing Helper on a beatmap without any redlines.
  • Fixed weird fullscreen behaviour of the main window.

Release version 1.6.8.0 (Github)

  • Hitsound Copier can now copy regular hitsounds to slidertick and sliderslide samples and it will automatically generate the required slidertick and sliderslide samples.
  • Any backups made by Mapping Tools now use the newest version from editor memory.
  • Added saving and loading of locked virtual objects in the project menu of Geometry Dashboard.
  • Added single point circle generator to Geometry Dashboard.
  • Added menu item that opens the Mapping Tools folder.
  • Added a 'New project' project menu item that resets any tool back to default settings.
  • Added indentation to config.json.
  • Increased default merge leniency in Slider Merger.
  • Fixed locked objects in Geometry Dashboard being impossible to snap to.
  • Fixed the lock hotkey accidentally removing too many locked virtual objects.
  • Fixed crash loop for people with corrupt registry.
  • Fixed tool's project possibly being null and breaking stuff.
  • Fixed .ogg Vorbis sample export not being compatible with weird sample rate source.
  • Fixed interactions on bezier sliders which end on a red anchor.

Release version 1.6.7.0 (Github)

  • Added a QuickUndo hotkey to the preferences that lets you instantly load the latest backup.
  • Added 'Load backup' menu item for loading backups to the current beatmap.
  • Added discernible beatmap backup filenames for periodic backups and manual backups.
  • Improved snapping behaviour in Geometry Dashboard when dragging objects.
  • Fixed backend not reading changed fields when using QuickRun.
  • Fixed break end time being set to the break start time.

Release version 1.6.6.0 (Github)

  • Property Transformer can now transform all storyboard events, break periods, and video offsets.
  • Property Transformer can now work on .osb files.
  • You can now automatically fetch hit object coordinates for new hitsound zones in Hitsound Preview Helper by shift-clicking the add button.
  • Success message of most tools is now displayed less intrusively.
  • Crash message now shows the whole path to the crash-log file.
  • Fixed Sliderator making bad paths.
  • Fixed crash on automatic osu! save override with BetterSave.
  • Fixed Hitsound Studio sometimes not converting non-Wave samples to Wave.

Release version 1.6.3.1 (Github)

  • Mapping Tools now actually runs in 64-bit mode.
  • Fixed greenline spam in Hitsound Copier when trying to conserve 5% volume greenlines with a map that only uses note specific sample volume.
  • Fixed problems when trying to export hitsounds in Hitsound Studio on coinciding mode while using the previous sample schema.
